Private Data Bridge (alpha)

Welcome to the alpha release of the Private Data Bridge.

The data bridge provides an API of services enabling users to syncronize, query, view, search and use AI services with their private user data. It also provides this developer sandbox interface, to easily interact with the private data bridge APIs.

Getting Started

Here's the recommended steps to get started:

  1. Create a Verida Network identity by scanning the QR code and obtain a private key.
  2. Connect some data to your identity such as Google which includes gmail. You may need to wait 10-15 minutes for your first backlog of data to syncronize. View the sync logs to track the progress.
  3. Try the private AI demo that has access to any data syncronized to your identity.

Create a Verida Network Account

You must have a Verida Network private key to use the Verida Private Data Bridge services. Your Verida Network key is used to unlock access to your encrypted data stored on the Verida Network. It's also used to encrypt / decrypt your data for use by applications. You can generate a private key by scanning this QR code and downloading the Verida Mobile Wallet

Once you have created an account on the Verida Mainnet (Myrtle), you can obtain your seed phrase by navigating to; Settings → Seed Phrase. Your seed phrase can be used anywhere you see Private Key in this developer sandbox.

It's also possible to create a Verida Network Identity using the Verida SDK Command Line Tools.

Security

This server is running within a Marlin Oyster secure enclave that protects all data and computation. This means that no-one, not even the underlying infrastructure provider for this server, can access any user data. See About → Security for more details.

For full transparency, the Verida Private Data Bridge is open source: https://github.com/verida/data-connector-server

Developer Sandbox (alpha)

You are currently viewing the developer sandbox, which is a basic web interface that interacts with the Verida Private Data Bridge APIs.

This is your entry point to the following services:

  1. Connections — Connect, Disconnect, Sync and View Logs of active data connections.
  2. Private AI — A basic proof of concept of a private AI prompt that can access your user data.
  3. Browse Data — A basic interface to view data syncronized from third party data connections, along with sync information.
  4. API Playground — An interactive API playground for developers who want to integrate their applications / AI agents with the private data bridge to access user data.
  5. About — Learn more about the technology, security and architecture powering this private data bridge.

Developer APIs

The private data bridge API's enable developers to securely integrate user data into their applications and AI agents::

  • /api/v1/db — View and Query user databases
  • /api/v1/ds — View and Query user datastores (databases that enforce structured data; ie: a database of emails)
  • /api/v1/minisearch — Powerful keyword search on user data
  • /api/v1/email — An advanced interface for searching user emails
  • /api/v1/chat — An advanced interface for searching user chat message history
  • /api/v1/llm — Access a basic, built-in AI prompt service. There is a generic prompt service and a personalized prompt service that incorporates user data into AI responses. Developers can fork this and build their own AI solutions that leverage user data.

You can learn how to integrate these API's into your own applications in the API Playground.
