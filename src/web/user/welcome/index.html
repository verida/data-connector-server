Private Data Bridge [alpha]

Welcome to the alpha release of the Verida Private Data Bridge.

The Private Data Bridge provides a set of API services for developers. These APIs support data syncronizing, queries, keyword searches and enables the development AI services with private user data.

This is a developer preview, enabling developers to learn about the available APIs and explore how best to integrate into their applictions.

There is also a simple RAG AI prompt interface (with API endpoint) as showcase of the types of AI solutions developers can build.

Getting Started

Here's the key steps to get started:

  1. Create a Verida Network account — You should have done this already if you are logged in and seeing this page
  2. Connect some data to your account such as Google which includes gmail. You may need to wait 10-15 minutes for your first backlog of data to syncronize. Click the "Sync logs" button to track the sync progress.
  3. Try the private AI demo that has access to any data syncronized to your identity.
  4. Explore the APIs available to developers, by querying your syncronized data.
